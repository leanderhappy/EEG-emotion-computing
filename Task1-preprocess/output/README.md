# 输出目录

这里保存任务一预处理生成的报告素材和运行汇总。

常见输出：

- `output/figures/<subject>/psd_comparison.png`：滤波前后频谱对比图
- `output/figures/<subject>/waveform_comparison.png`：清洗前后波形对比图
- `output/figures/<subject>/ica_eog_scores.png`：ICA 成分与眼电通道相关分数图
- `output/figures/<subject>/ica_eog_topomap.png`：疑似眼电 ICA 成分头皮分布图
- `output/task1_run_summary.csv`：本次批处理运行汇总

图片和汇总表由脚本生成，仓库只保留目录说明。
