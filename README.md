# 情感脑电信号处理与识别系统

本仓库是《人机交互选修课》大作业的实现代码，采用 **纯 Python 技术路线**。本项目通过 MNE-Python 库实现对脑电数据（如 DEAP/MAHNOB-HCI）的预处理、特征提取及情感分类任务，并包含了迁移学习/主动学习的对比实验。

## 🛠️ 任务完成情况

### 任务 1：脑电信号预处理 
- **数据导入**：支持 `.mat` 或 `.fif` 格式数据转换。
- **信号滤波**：带通滤波去除工频干扰。
- **伪迹去除**：利用 ICA（独立成分分析）去除眼电与肌电伪迹。
- **分段与校正**：以视频刺激为基准切分数据，并进行基线校正。

### 任务 2：情感计算基础建模
- **特征提取**：提取时域（均值、方差）与频域（功率谱密度 PSD）特征。
- **分类器**：基于 `scikit-learn` 实现 SVM 与 逻辑回归，进行效价（积极/消极）二分类。

### 任务 3：算法优化（二选一）
- **方案 A - 迁移学习**：对比跨被试情感识别的准确率差异。
- **方案 B - 主动学习**：对比随机采样与主动学习在 10%~20% 标注数据下的性能。

### 任务 4：结果分析与应用展望 
- 输出准确率、精确率、召回率及混淆矩阵。
- **应用设想**：基于脑电情感识别的自适应音乐推荐系统方案设计。
